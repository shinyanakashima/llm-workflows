# 業務文脈 / ビジネスコンテキスト

## 会社概要

- 北海道帯広市にある建設コンサルタント「ズコーシャ」。
- 建設コンサルでありながら IT 事業部を持つ。
- 道東でデータセンターを長年運営してきた実績があり、自治体や民間のデータを預かりながら、システム運用や導入支援を行っている。

## 組織と役割

### IT 事業部 イノベーション課

- 地理空間情報を戦略的な技術とした新規サービス開発を担当。
- オープンイノベーションの考え方を取り入れながら、新規事業の創出・サービス化・収益化を目指す。

### 事業変革室（全社横断・長期プロジェクト）

- 「外部との共創」をミッションとする。
- コレクティブ・インパクトによる社会課題解決を目指しながら、将来的にビジネスとしても成り立つ事業の創出を目指す。

## 提案・企画検討で両立する2つの視点

### IT 事業部の視点

- 新規事業創出 / サービス化 / 収益化
- 技術的実現性
- 運用可能性

### 事業変革室の視点

- 外部との共創 / オープンイノベーション / コレクティブ・インパクト
- 地域課題解決
- 中長期的な関係づくり
- 将来的な事業接続

## 活かすべき強み

- 地理空間情報を戦略技術として活用できること。
- 農業・環境・まちづくりと IT を組み合わせた総合力。
- 自治体・民間データを預かるデータセンター運用実績。
- システム運用・導入支援の実績。

## 提案・企画検討時の基本姿勢

単なる技術提案ではなく、以下の観点を含める。

- 地理空間情報を戦略技術として活用できるか
- 農業・環境・まちづくり・地域課題と接続できるか
- ズコーシャの既存の強みと接続できるか
- IT 事業部の新規事業・収益化につながる可能性があるか
- 事業変革室の共創・中長期活動として意味があるか
- 外部機関との関係づくりにつながるか
- 実証実験・共同研究・補助金活用の余地があるか
- 地域への社会的インパクトと、将来的なビジネス成立性を両立できるか
