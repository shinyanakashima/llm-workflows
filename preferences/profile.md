# プロフィール / 役割

## 所属と役割

- 北海道帯広市にある建設コンサルタント「ズコーシャ」に勤務。
- IT 事業部 イノベーション課の課長として、地理空間情報を戦略的な技術とした新規サービス開発を担当。
- 全社横断の長期プロジェクトである「事業変革室」にも所属。

## 担当領域

- イノベーション課では、オープンイノベーションの考え方を取り入れながら、新規事業の創出・サービス化・収益化を目指す。
- 事業変革室では、「外部との共創」をミッションとし、コレクティブ・インパクトによる社会課題解決を目指しながら、将来的にビジネスとしても成り立つ事業の創出を目指す。

## 提案・企画検討で常に両立する2つの視点

### IT 事業部の視点

- 新規事業創出
- サービス化
- 収益化
- 技術的実現性
- 運用可能性

### 事業変革室の視点

- 外部との共創
- オープンイノベーション
- コレクティブ・インパクト
- 地域課題解決
- 中長期的な関係づくり
- 将来的な事業接続

## 強みとして活かす要素

- 地理空間情報を戦略技術として活用する。
- 農業・環境・まちづくりと IT を組み合わせた総合力を活かす。
- 道東でのデータセンター運営実績、システム運用・導入支援の実績を活かす。
