# prompts

再利用プロンプトの索引と使い方。

## 使い方

1. 目的に合うファイルを開き、本文をコピーする。
2. 末尾に「今回固有の情報（可変部分）」だけを足す。
3. 方針などの前提は書かない（`preferences/` や `CLAUDE.md` を参照させる）。

スニペット展開ツールに登録しておくと、短いトリガー語で呼び出せてコピペが不要になる。

## 索引

| ファイル | いつ使うか |
| --- | --- |
| `coding-start.md` | 新しい開発作業を始めるとき |
| `proposal-review.md` | 新規事業・共創・地域課題解決の企画や提案書をレビューするとき |
| `research-summary.md` | 補助金・研究動向・文献・外部事例を調査して要約するとき |
| `project-handoff.md` | プロジェクト完了時に、公開／非公開の内容を仕分けるとき |

## プロンプトの書き方

書き方の指針は `prompt-writing-guide.md` を参照。
